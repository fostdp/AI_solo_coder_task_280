# 🧠 神经生理学虚拟实验室

一个功能完整的交互式神经生物学学习工具，通过Canvas可视化展示神经元电生理现象。

## ✨ 功能特性

### 🎯 基础动作电位
- **静息电位**：膜电位约-70mV的稳定状态
- **去极化**：Na⁺通道开放，Na⁺内流，膜电位上升
- **复极化**：K⁺通道开放，K⁺外流，膜电位下降
- **超极化**：K⁺通道延迟关闭，膜电位短暂低于静息水平
- **实时电位变化图**：动态显示膜电位变化曲线
- **离子流动画**：可视化Na⁺和K⁺的跨膜运动

### ⚡ 有髓鞘跳跃传导
- **朗飞氏结**：动作电位只在结处产生
- **髓鞘绝缘**：减少离子泄漏，提高传导速度
- **跳跃式传导**：电信号在结间快速传递
- **速度对比**：传导速度提高约50-100倍
  - 无髓鞘轴突：0.5 - 2 m/s
  - 有髓鞘轴突：20 - 120 m/s

### 🧬 突触传递模拟
- **突触前神经元**：动作电位传导到轴突末梢
- **神经递质释放**：囊泡与膜融合释放递质
- **突触间隙**：递质扩散到突触后膜
- **受体结合**：递质与突触后膜受体结合
- **突触后电位**：突触后神经元产生电位变化

### ☠️ 药物影响模拟（河豚毒素）
- **TTX特异性阻断**：阻断电压门控Na⁺通道
- **动作电位抑制**：无法产生完整动作电位
- **对比观察**：正常与阻断状态下的电位变化差异
- **洗脱功能**：模拟药物洗脱恢复正常功能

### 🔬 膜片钳虚拟实验
- **千兆欧封接**：电极与细胞膜形成高阻封接
- **封接过程动画**：电阻从0逐步增加到1GΩ
- **全细胞模式**：破膜获得全细胞记录
- **单通道电流记录**：实时记录离子通道电流
- **数据可视化**：实时显示记录到的电流信号

### 📊 学习进度追踪
- **8个学习步骤**：从基础到高级全面覆盖
- **进度自动保存**：localStorage和后端双重存储
- **步骤标记**：完成步骤自动标记

## 🧪 科学原理

### 静息电位（-70mV）
- 细胞膜对K⁺通透性较高
- Na⁺-K⁺泵维持离子梯度（3Na⁺出，2K⁺入）
- 膜内负电位，膜外正电位

### 动作电位产生
1. **去极化**：刺激达到阈值（-55mV），Na⁺通道开放
2. **峰值**：膜电位达到约+40mV，Na⁺通道失活
3. **复极化**：K⁺通道开放，K⁺外流
4. **超极化**：K⁺通道延迟关闭，膜电位低于静息水平

### 跳跃传导优势
1. 节能：只在朗飞氏结处有离子跨膜运动
2. 快速：电信号在结间被动快速传导
3. 准确：传导更可靠，不易衰减

### 河豚毒素（TTX）作用机制
- 特异性结合电压门控Na⁺通道的外口
- 物理阻断Na⁺内流
- 不影响K⁺通道功能
- 可用于研究离子通道特性

### 膜片钳技术
- **封接**：玻璃微电极与细胞膜形成高阻封接（GΩ级）
- **全细胞记录**：破膜后记录整个细胞的电活动
- **单通道记录**：可记录单个离子通道的电流（pA级）
